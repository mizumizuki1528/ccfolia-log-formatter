# ココフォリア ログ整形ツール

ココフォリア（CCFOLIA）から出力されるセッションログHTMLを、読みやすい形式に整形・編集・エクスポートするブラウザベースのツール。

## 使い方

1. `index.html` をブラウザで開く
2. ココフォリアのログHTMLをドロップ or 選択
3. シナリオタイトルとKP（ナレーター）を設定
4. ツールバーから編集・色設定・タブ設定などを操作
5. エクスポートボタンで整形済みHTMLをダウンロード

エクスポートされたHTMLには編集状態が埋め込まれており、再度ドロップすると続きから編集できます。

## ファイル構成

```
ccfolia_log_formatter/
├── index.html         エントリーポイント
├── style.css          スタイル
├── parser.js          ココフォリアHTMLパーサー
├── app.js             メインアプリケーションロジック
├── i18n.js            日本語UIラベル適用
├── samples/           サンプルログ
│   ├── 11_elf_all.html         日本語タブ名のサンプル
│   ├── 群青とエトワ[all].html  英語タブ名のサンプル
│   └── reference/              参考: 別ツールでの整形例
├── README.md          本ファイル
└── 作業ログ.md        開発作業ログ
```

## 主な機能

- 発言の編集・削除・並び替え
- 発言者ごとの色・アイコン・地文（ナレーション風表示）切替
- タブごとの種類（メイン/雑談/情報/秘匿）と背景色設定
- 章の自動検出と手動追加・ドロップダウンナビ
- 検索（HTML構造保持ハイライト）
- 雑談を右サイド注釈として時系列配置
- 秘匿の連続発言をブロック化
- 情報タブをノート風カードで表示
- 編集状態をHTMLに埋め込んで再編集可能
- 日本語/英語タブ名両対応

## 技術メモ

- 単一HTML/CSS/JSで動作（依存関係なし）
- HTMLファイルはBOM付きUTF-8で保存（文字化け回避）
- 日本語UIは `i18n.js` でDOM読み込み後に適用
