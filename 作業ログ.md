# ココフォリア ログ整形ツール 作業ログ

## プロジェクト概要

ココフォリア（CCFOLIA）から出力されるセッションログHTMLを、読みやすい形式に整形・編集・エクスポートするブラウザベースのツール。

## ファイル構成

```
ccfolia_log_formatter/
├── index.html         エントリーポイント（HTML構造、ASCIIのみ）
├── style.css          スタイル定義（ライト/ダーク対応）
├── parser.js          ココフォリアHTML→構造化データ変換
├── app.js             UI制御・編集・エクスポートロジック
├── i18n.js            DOM読み込み後に日本語UIを適用
├── 11_elf_all.html    サンプルログ（ココフォリア出力 / 日本語タブ名）
├── 11-elf-_main.html  別ツールでの整形例（参考）
├── 群青とエトワ[all].html  サンプルログ（英語タブ名）
└── 作業ログ.md        本ファイル
```

## 実装済み機能

### 基本機能
- ログHTML読み込み（ドラッグ＆ドロップ / ファイル選択 / 複数ファイル結合）
- 編集済みHTMLの再読み込み（埋め込みJSONデータから状態復元）
- KP/ナレーター選択画面
  - シナリオタイトル入力欄付き（デフォルト：ファイル名から `[xxx]` を除いた名前 + `_編集済み`）
  - ファイル再選択ボタン
- ヘッダー＆ツールバー sticky 配置
- シナリオタイトルをヘッダー、ブラウザタブ、エクスポートファイル名に反映
- 検索窓（リアルタイムフィルタ＋HTML構造保持ハイライト）
- タブ別フィルター（ボタン選択）
- 章（話）の自動検出とドロップダウン形式ナビ

### 編集機能
- 発言の編集（発言者名・内容・色・地文フラグ）
  - タブ名を編集モーダル上部に表示（参考情報）
  - 改行を `<br>` ⇔ `\n` で正しく相互変換
- 発言の削除
- 発言の並び替え（ドラッグ＆ドロップ）
- 章区切りの追加（各発言操作の?ボタン）
- 章区切りの削除（章ヘッダーホバーで×ボタン）

### タブ設定パネル（独立）
- 各タブの種類選択（メイン / 雑談 / 情報 / 秘匿）
- 各タブの表示ON/OFF切替（旧 Secret: ON/OFF を統合）
- 情報・秘匿タブの背景色設定
- 任意のタブ名（独自タブ）に対応可能

### 色・アイコン管理（カラーパネル）
- 発言者ごとの色一括変更
- 発言者ごとのアイコン画像設定（Data URL埋め込み）
- 発言者ごとの「地文」トグル（ナレーション風表示の切替、KP指定の代替）

### エクスポート
- 整形済みHTMLファイルとしてダウンロード（BOM付きUTF-8）
- 単体HTMLで完結（アイコンはData URL埋め込み）
- 編集状態を `<script type="application/json" id="ccfolia-data">` に埋め込み、再読み込みで復元可能
- ファイル名はシナリオタイトルベース
- ヘッダーにシナリオタイトルとドロップダウン章ナビを sticky 配置
- 雑談右サイド配置時は float-right で時系列を保ちつつメインに空白を作らない

### エクスポートのレイアウト
- **メイン（PC会話）** → 角丸吹き出し、白背景、アイコン付き
- **メイン（地文）** → 全幅、吹き出しなし、地の文風
- **秘匿（PC）** → 角丸吹き出し（タブカラー薄め背景＋破線左ボーダー）
- **秘匿（地文）** → ナレーション風（タブカラー背景＋破線左ボーダー）
- **秘匿の連続** → 同じ秘匿タブの連続発言は1つの背景ブロックにグループ化
- **雑談** → 右サイドに float-right 配置、シンプルな左ボーダーのみ
- **情報** → カード風ノート（黄色背景＋金色ボーダー＋影、発言者名をタイトル化）

## ビューア表示
- 発言者名の下にタブ名併記（メインのみ非表示で視覚ノイズ抑制）
- 秘匿は `[secret(...)]`、情報は `? info`、雑談は `[other]` 等

## 言語対応

### タブ名認識
- 日本語: `メイン` `雑談` `情報` `秘匿(...)`
- 英語: `main` `other` / `chat` `info` / `information` `secret(...)`

### 章検出
- 日本語: `N話 タイトル`
- 英語: `Chapter N: title` / `Episode N: title` / `Scene N: title`

## 技術的な工夫

### 文字エンコーディング対応
- HTMLファイルはBOM付きUTF-8で保存（一部環境での文字化け回避）
- HTMLには日本語を直接書かず、`i18n.js` でDOM読み込み後にUnicodeエスケープで日本語ラベルを適用

### パーサー（parser.js）
- ココフォリアの `<p style="color:...;">` 構造をパース
- タブ名・発言者・内容・色を抽出
- 秘匿タブは `^秘匿` `^secret` パターンで検出
- ダイスロール結果を検出
- KP/ナレーター発言から章マーカーを自動生成
  - 自動マーカーは再検出時にリセット、手動マーカー（`source: 'manual'`）は保持

### レイアウト技法
- 雑談右配置に `float: right; clear: right;` を使用
  - メイン側に縦の空白を作らず、雑談ブロックを避けてメインがフローで詰まる
- 秘匿の連続発言は `border-radius` を境界部分のみに適用してブロック感を演出
- ヘッダー・ツールバーは `position: sticky` で常時表示

### データ復元
- エクスポートHTML末尾に編集状態をJSON埋め込み
  - `<script type="application/json" id="ccfolia-data">{...}</script>`
  - スクレイプ防止のため `</script>` を `<\/script>` にエスケープ
- 単一ファイルで埋め込みデータが見つかった場合、即座に復元してKP選択画面をスキップ

## 追加実装ログ（7回目セッション）

### 実施内容

#### #2 白い色の視認性対応
- テキスト色と背景色のコントラスト比を計算（WCAG準拠）
- コントラスト比 1.8:1 未満の場合のみ縁取りを自動付与
- 背景が明るければ黒縁取り、暗ければ白縁取り
- 発言者ごとに手動で縁取り設定可能（自動/なし/黒/白）

#### 背景色変更機能
- ツールバーに背景色カラーピッカーを追加
- エクスポート時に全体の配色を背景色に応じて自動調整
  - 地の文、吹き出し、章見出し、区切り線、ヘッダー、リンク等
- 暗い背景では情報タブの透過度を25%に上げて視認性確保
- 保存データに含まれ、再読み込み時にUIに復元

#### #6 ヘッダーカラー設定（グラデーション対応）
- ツールバーにヘッダーカラーピッカー3つ追加（色1、色2、テキスト色）
- 2色設定 → `linear-gradient(to bottom, 色1, 色2)` で縦グラデーション
- 1色のみ → 単色背景
- 未設定 → 背景色に応じた自動配色
- テキスト色 → タイトルとリンクの色に反映
- 保存データに含まれ、再読み込み時に復元

#### ダイスカラー調整
- 失敗系: `#ff00ff`（マゼンタ）→ `#ff4444`（赤）に変更
- 検出対象追加: 決定的成功、決定的成功/スペシャル、致命的失敗
